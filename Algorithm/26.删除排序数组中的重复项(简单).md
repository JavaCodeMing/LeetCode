题目;
    给定一个排序数组,你需要在"原地"删除重复出现的元素,使得每个元素只出现一次,返回移除后数组的新长;
    不要使用额外的数组空间,你必须在"原地"修改输入数组,并在使用O(1)额外空间的条件下完成;
    示例:
        给定数组 nums = [1,1,2], 
        函数应该返回新的长度2,并且原数组 nums 的前两个元素被修改为 1, 2;
        你不需要考虑数组中超出新长度后面的元素。
1.双指针:
    [1]思路: 
        (1)定义两个指针,一个用来对不重复的值进行保存,另一个用来遍历数组;
        (2)遍历数组的指针一旦发现不重复,则将值给保存不重复值的指针的位置,且该保存指针右移一位;
    [2]实现:
        class Solution {
            public int removeDuplicates(int[] nums) {
                if (nums.length == 0) {
                    return 0;
                }
                // 定义一个指针用来对不重复的值进行保存
                int i = 0;
                for (int j = 0; j < nums.length; j++) {
                    // 判断遍历指针对应的值与前一个不重复值是否相等
                    if (nums[i] != nums[j]) {
                        nums[++i] = nums[j];
                    }
                }
                // 最终数组的长度为最后一个不重复值的下标加1
                return i + 1;
            }
        }
    [3]复杂度分析:
        (1)时间复杂度: O(N),N为数组的长度
        (2)空间复杂度: O(1)

题目:
    给你一个数组nums和一个值val,你需要"原地"移除所有数值等于val的元素,并返回移除后数组的新长度;
    不要使用额外的数组空间,你必须仅使用 O(1) 额外空间并"原地"修改输入数组;
    元素的顺序可以改变,你不需要考虑数组中超出新长度后面的元素;
    示例:
        给定 nums = [3,2,2,3], val = 3,
        函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2;
        你不需要考虑数组中超出新长度后面的元素;
1.双指针:
    [1]思路:
        (1)定义两个指针,一个用来保存不等于目标值的数,另一个用来遍历数组;
        (2)遍历数组的指针对应值不等于目标值,则交给保存不等于目标值的指针的位置,且该保存指针右移一位;
    [2]实现:
        class Solution {
            public int removeElement(int[] nums, int val) {
                if (nums.length == 0) {
                    return 0;
                }
                // 定义一个指针用来处理不重复的值
                int i = 0;
                for (int j = 0; j < nums.length; j++) {
                    // 判断遍历指针对应的值与
                    if (nums[j] != val) {
                        nums[i++] = nums[j];
                    }
                }
                return i;
            }
        }
    [3]复杂度分析:
        (1)时间复杂度: O(N),N为数组的长度
        (2)空间复杂度: O(1)
